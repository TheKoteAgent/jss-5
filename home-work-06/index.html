ПРАЦЮЄМО З ФОРМАМИ: input, range, textarea, checkbox, radio

1 Перевірка стану checkbox

При натисканні кнопки #b-1 запускається функція f1. Функція перевіряє стан #i-1 і якщо він checked - виводить true, якщо не вибрано - false. Виведення #out-1.

2 Отримання value з checkbox

При натисканні кнопки #b-2 запускається функція f2. Функція перевіряє стан #i-2 і якщо він checked – виводить value даного елемента, якщо обраний - false. Виведення #out-2.

3 Перевірка довжини пароля

При натисканні на кнопку виводьте в #out-31 value, прописане в input #i-3. У #out-32 виводьте 1 якщо довжина пароля більше або дорівнює 6 або 0, якщо менше. Для підрахунку кількості символів у рядку використовуйте length.

4 Робота з radio кнопками

При натисканні кнопки #b-4 запускаємо функцію f4. Функція перевіряє checked елементів. Якщо елемент вибраний (checked) виводить в #out-4 value вибраної радіо кнопки.

5 Робота з color picker

При натисканні кнопки #b-5 запускається функція f5. Функція отримує колір з #i-51 і надає як value елементу #i-52. Тобто, після натискання кнопки вибраного кольору в першому і другому input стануть однакові.

6 Робота з date input

При натисканні кнопки #b-6 запускається функція f6. Функція виводить дату з #i-6 у #out-6.

7 Робота з range input

9 Робота з select

При зміні вибраного значення #s-9 виводьте в #out-9 value вибраного option з #s-9.
